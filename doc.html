Documentation for qlik-app-reloader

Running the software: in a Powershell CD into the folder where the exe is located and you can run 

            ./reloadAllApps.ps1

Rebuild after changes to main.go file with 

            GOOS=windows go build -o qlik-app-reloader.exe main.go
